Tratamento
----------
Juntei dados de diferentes bases para criar um diretório de compatibilização. Em maior detalhe, peguei dados de identificação de todas as escolas 
da BigQuery da Base dos Dados e fiz um left_join com os dados tratados provenientes da Secretaria de Educação de São Paulo. O join foi feito pela
id_escola, variável que identifica um código para cada escola a nível nacional.
